本项目为计算机视觉与数据库结合的智慧交通项目
本项目实现的功能主要有以下几点：
1.利用视觉识别七种不同类型的车辆
2.利用deepsort算法跟踪车辆
3.算法设计实现拥堵预测以及判断
4.算法设计车速判断
5可供用户用图片，图片集，视频，摄像头进行使用
6.将收集到的一系列数据，如当前时间，车辆种类，画面中的车辆总数，车速，当前拥堵情况，未来拥堵情况等等，发送至数据库供前端使用
